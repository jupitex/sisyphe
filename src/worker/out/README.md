sisyphe-out
========
Output module for sisyphe analyser

Will export data to a json file


## Output formats
By default, this module will export the transferred data to 
- JSON file


## JSON file
The JSON file will be located in : 
`/sisyphe/out/{timestamp}-sessionName.json`
